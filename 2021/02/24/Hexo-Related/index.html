发表于 2021-02-24 更新于 2021-03-12 阅读次数： Valine：

博客搭建在Hexo和NextT上面，七七八八花在上面的时间着实不少，但是并没有记录下来。这里开一个博客来记录一下。

目前的版本如下：

昨天刚刚升级哒，(づ￣ 3￣)づ，开心，我觉得新版本还是不错的

升级的过程

Hexo版本升级指南

  • 里面的一些命令执行不起来，大约是我的npm环境有些问题

  • 执行npm命令时候带上了参数--loglevel verbose，不然屏幕上面啥输出都没有

  • 成功执行的命令有：

    npm i hexo-cli -g  #这条命令之后hexo-cli就升级好了
  npm install -g npm-upgrade  #安装了npm-upgrade
  npm update -g
  npm update --save

  • NextT的升级简单直接多了，直接进行git clone就好了。

Hexo Next主题的评论功能

  • 目前选择使用的是valine，这个也是已经在NextT中有所集成的，同样需要在leanCloud上面有张表，需要的表名是Comment。

阅读数量

Hexo博客Next主题添加统计文章阅读量功能

  • 作为一个孤独地默默地写着博客的人，对于“我的博客有人看吗？”这种疑问，绝对是好奇的，所以阅读数量也很关键啊，当然，也可能受到一万点的暴击，没人看，只能自己点吧 😦
  • 更新：果然受到了一定打击，没人看啊。。。只能收拾好心情，自己加油。安慰自己一下，咳，就当写日记吧
  • 参考了上面一篇文档，但是其中还是遇到了一些问题。NextT集成的还是不错的，配置leancloud_visitors就可以了。
  • 碰到的问题和解决方法：
    • leancloud 国内版在注册后需要实名认证了，绕过去可以访问国际版leancloud.app，这样就不需要提供身份证号码。手机验证码还是要的
    • Counter not initialized! More info at console err msg. 出现这个错误的原因是由于没有安装hexo-leancloud-counter-security插件，并且在配置中将leancloud_visitors.security选项设成了true的缘故。
    • 没有数据。这个是因为在leancloud上面建的数据表名字必须严格为Counter，我一开始建了表名为counter，结果就找不到数据了
