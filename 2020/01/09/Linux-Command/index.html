CHMOD

  • 表示用户组的数字：
    • 一般是三个数字：
      • 第一个数字表示文件所有者的权限
      • 第二个数字表示与文件所有者同属一个用户组的其他用户的权限
      • 第三个数字表示其它用户组的权限。
  • 读（r=4），写（w=2），执行（x=1）
  • T 代表目录只能由当前目录的owner来修改

AWK

  • linux awk命令详解
  • 使用awk分割字符串并且获取分割后的最后一个字符串

Example

awk 其实是读入一行，进行查找，再进行操作。可以集合SED和GREP的作用。下面用例子来说明问题

  • awk -F'-' '{print $1}'
    • 用-做分隔符，然后将-分隔之后的第一列数据输出出来。
    • 如果有版本，比如2.0.1-SNAPSHOT,这个语句就可以得到2.0.
  • echo "you-me-he" |awk -F '[-]' '{print $NF}'
    • 这条命令是用-分隔之后，打印分隔后的最后一个字符，所以输出he

判断文件

  • shell判断文件存在

实例

  • 判断文件存在与否

    if [ ! -f “/data/filename” ];then
    echo "文件不存在"
    else
    rm -f /data/filename
    fi

  • 判断目录存在与否：将-f换成-d

if [ -d "/data/" ];then
echo "文件夹存在"
else
echo "文件夹不存在"
fi

字符串操作

截取字符串的方法

  • Shell脚本8种字符串截取方法总结

实例

  • var=http://www.aaa.com/123.htm
  • echo ${var#*//}
    • # 号是运算符，*// 表示从左边开始删除第一个 // 号及左边的所有字符
    • 所以这样的话会将http://都给删除掉
  • echo ${var##*/}
    • ##是运算符指最后一个，*/表示/和/左边的所有字符。所以一个#是第一个，两个是最后一个。
    • 这句话会删除最后一个/以及左边的字符， 结果只留下了123.htm

AIDE

  • AIDE is a file integrity checking tool。
  • Linux高级入侵检测 AIDE(Adevanced Intrusion Detection Environment,高级入侵检测环境)是个入侵检测工具，主要用途是检查文本的完整性。
  • AIDE还能够使用下列算法：sha1、md5、rmd160、tiger，以密文形式建立每个文档的校验码或散列号
